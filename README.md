# Doctor Appointment Booking System

## 📌 Abstract
The Healthcare Appointment Management System is a web application designed to simplify doctor appointment booking. Users can search doctors, view available slots, and book appointments easily. It reduces waiting time and improves scheduling efficiency.

## 🚀 Features
- Doctor Search & Filter
- Appointment Booking
- Appointment Management
- Responsive UI

## 🛠️ Tech Stack
- React.js
- HTML, CSS, JavaScript
- React Router
- LocalStorage
